# Vision Env Checker - 工业视觉运行环境检测工具

A cross-platform environment checker for industrial machine vision systems.

## 功能特性

- 🔍 硬件检测：GPU、CPU、内存、磁盘
- 📷 相机驱动检测：Basler、海康、大华等
- 🌐 网络连接检测
- 🔧 软件环境检测：CUDA、OpenCV、Halcon 等
- 📊 一键生成检测报告
- ⚡ 自动修复建议

## 技术栈

- Python 3.10+
- PyQt6 / PySide6 (GUI)
- psutil (系统信息)
- wmi (Windows 硬件信息)

## 快速开始

```bash
pip install -r requirements.txt
python src/main.py
```
